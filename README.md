# 🧠 IA-Powered CV Creator

¡Generá tu currículum Fácil y Rápido! Esta es una plataforma moderna y minimalista que utiliza Inteligencia Artificial para transformar tu trayectoria en un CV profesional de alto impacto en cuestión de segundos.

> **Estado del Proyecto:** 🚧 En Desarrollo (Maquetación y lógica de UI)

---

## ✨ Funcionalidades Principales

*   **Generación Inteligente:** Redacción automática de experiencia y logros basada en tus datos.
*   **Live Preview:** Visualización en tiempo real del diseño de tu CV mientras completas el formulario.
*   **Optimización de Texto:** Mejora tus descripciones con un solo clic usando IA para sonar más profesional.
*   **Diseño Moderno:** Interfaz limpia, responsiva y enfocada en la conversión (UX).
*   **ATS Friendly:** Estructura de código optimizada para pasar los filtros automáticos de las empresas.

## 🧩 Tecnologías Usadas

Este proyecto utiliza las herramientas más modernas del ecosistema web:

*   **Frontend:** [Next.js 15](https://nextjs.org) (App Router) + [React](https://reactjs.org)
*   **Estilos:** [Tailwind CSS](https://tailwindcss.com)
*   **Lenguaje:** [TypeScript](https://typescriptlang.org)

## 🛠️ Instalación y Uso Local

1. Clona el repositorio:
   ```bash
   git clone https://github.com
   ```
2. Entra a la carpeta:
   ```bash
   cd IA-Powered-CV-Creator
   ```
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Inicia el servidor de desarrollo:
   ```bash
   npm run dev
   ```
5. Abre [http://localhost:3000](http://localhost:3000) en tu navegador.
